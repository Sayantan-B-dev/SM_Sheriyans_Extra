# 14. Login Controller Logic
➡️ Go to auth.controller.js
➡️ Make login function where:
   - Take email and password from body
   - Check with email in database for existing user
   - If not user tell something wrong with 400 status code
   - Now check is password valid with bcrypt
   - If not valid send the same 400 status message
   - Create a token with jwt.sign and user id, user role, jwt secret, and expiresIn
   - Save token in cookie
   - In the end send status with 200, message and user info
