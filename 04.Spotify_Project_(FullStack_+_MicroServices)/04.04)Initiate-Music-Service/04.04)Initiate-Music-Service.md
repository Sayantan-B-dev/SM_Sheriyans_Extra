```markdown
# 1. Create Service Folder
➡️ create a folder `/music-service`
```

```markdown
# 2. Initialize Project & Install Dependencies
➡️ npm init -y  
➡️ npm i express mongoose jsonwebtoken cookie-parser morgan dotenv
```

```markdown
# 3. Create App Entry File
➡️ create `/src/app.js`
```

```markdown
# 4. Enable ES Modules
➡️ change `"type"` to `"module"` in `package.json`
```

```markdown
# 5. App Setup
➡️ import `express` in `app.js`  
➡️ create an express app instance  
➡️ export the app instance
```

```markdown
# 6. Create Server File
➡️ create `server.js` in root  
➡️ import app  
➡️ listen on port `3000`
```

```markdown
# 7. Environment Configuration
➡️ create `/src/config/config.js`  
➡️ import `config as dotenvConfig` from `dotenv`  
➡️ call `dotenvConfig()`

➡️ create an object `_config` containing environment variables, for example:
   - `MONGO_URI: process.env.MONGO_URI || 'mongodb://localhost:27017/music'`
   - also include things like `JWT_SECRET`, etc.  
   - more fields can be added later as the project grows

➡️ export using `Object.freeze(_config)`
   // Object.freeze makes the config object immutable,
   // so values cannot be accidentally changed at runtime
```

```markdown
# 8. Database Connection
➡️ create `/src/db/db.js`  
➡️ import `mongoose` and `config`  
➡️ create a `connectDB` function that connects using `config.MONGO_URI`  
➡️ export the function
```

```markdown
# 9. Connect Database in Server
➡️ in `server.js`, import the database connection function  
➡️ call it before starting the server
```

```markdown
# 10. Environment Variables File
➡️ create `.env`  
➡️ put all required environment variables there
```

```markdown
# 11. Package Scripts
➡️ add `start` and `dev` scripts in `package.json`
```

```markdown
# 12. Music Model
➡️ create `src/models/music.model.js`  
➡️ import `mongoose`

➡️ create `musicSchema` with fields:
   - `title`
   - `artist`
   - `artistId` (not required, but if present it should be of type `ObjectId`)
   - `musicURL`
   - `coverImageURL`
   - enable `timestamps`

➡️ create the model  
➡️ export the model
```

```markdown
# 13. Music Routes
➡️ create `src/routes/music.routes.js`  
➡️ import `express`  
➡️ create an `express.Router()` instance  
➡️ export the router
```

```markdown
# 14. Register Routes in App
➡️ in `app.js`:
   - import `musicRoutes`
   - import `cookie-parser`
   - use `express.json()` middleware
   - use `cookieParser()` middleware
   - mount routes at `/api/music`
```
