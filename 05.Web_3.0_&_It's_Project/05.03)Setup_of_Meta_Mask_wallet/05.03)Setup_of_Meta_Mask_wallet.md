```markdown
### ▶ MetaMask Installation
➡️ go to **Chrome Web Store**
➡️ search **MetaMask Wallet**
➡️ install the extension
```

```markdown
### ▶ Wallet Setup
➡️ open MetaMask
➡️ create a new wallet **or** login with an existing one
➡️ complete the initial setup (password / recovery phrase)
```

```markdown
### ▶ Enable Test Networks
➡️ open MetaMask
➡️ click the hamburger menu (☰)
➡️ go to **Settings**
➡️ open **Advanced**
➡️ scroll down
➡️ turn ON **Show test networks**
   // required to access Ethereum testnets like Sepolia
```

```markdown
### ▶ Select Network
➡️ click the network dropdown (usually shows “Ethereum Mainnet”)
➡️ switch to **Custom / Test Networks**
➡️ select **Sepolia**
```

```markdown
### ▶ Copy Wallet Address
➡️ under **Tokens**, make sure Sepolia is selected
➡️ copy your **ETH address**
   // this is your public wallet address
```

```markdown
### ▶ Get Test ETH (Faucet)
➡️ go to: https://sepolia-faucet.pk910.de/
➡️ paste your copied ETH address
➡️ submit to receive test ETH
   // faucet sends free test ETH for development and testing
```

```markdown
### ▶ Result
➡️ test ETH will appear in your MetaMask wallet
➡️ you can now:
   - deploy smart contracts
   - pay gas on Sepolia
   - test Ethereum applications safely
```
